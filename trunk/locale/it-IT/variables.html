Tag speciali - semplice :
%asf_D%= Nome dominio. (senza http://)
%asf_F%= Nome file con estensione.
%asf_Fx%= Nome file senza estensione.
%asf_d%= Campo "Dominio" dal filtro usato. (I caratteri non permessi verranno rimossi. Se nessun filtro viene trovato, viene usato il dominio corrente)
%asf_f%= Campo "Nome file" dal filtro usato. (I caratteri non permessi verranno rimossi. Se nessun filtro viene trovato, viene usato il dominio corrente)
%asf_x%= Estensione file. (Senza il punto)

Tag speciali - avanzate :
%asf_rd%Espressione regolare%asf_rd%
Controlla il dominio completo con l'espressione regolare contenuta tra i due tag %asf_rd% e ritorna i dati presi.

%asf_rf%espressione regolare%asf_rf%
Controlla il nome del file con l'espressione regolare contenuta tra i due tag %asf_rf% e ritorna i dati presi.

ATTENZIONE : Se il risultato per il percorso di salvataggio contiene caratteri non permessi(i : or i ?"*\/<>|: su Windows), l'estensione smetterà di funzionare. Controllare i filtri se l'estensione smette di funzionare.
