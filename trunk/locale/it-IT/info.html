Automatic Save Folder V0.9.4b
by Cyan

Importante:

Questa estensione funziona solamente se si usa la finestra di download (quella che chiede di aprire, eseguire o salvare file su disco)

Se Firefox non mostra tale finestra, l'estensione non funzionerà.


Se qualche filtro non funziona, accertartati che Firefox non usi un altro processo sul file scaricato:

1.
   Vai sul menu "Strumenti" e "Opzioni...".
2.
   - Per utenti di Firefox 3: vai sul tab "Applicazioni"
   - Per utenti di precedenti versioni di Firefox: Vai sul tab "Contenuti" e clicca su "gestisci..." in basso a destra.
3.
   Verifica se il file che vuoi scaricare sia impostato su "Chiedi ogni volta".







Come si usa:


Questa estensione permette di definire, usando dei filtri, quale cartella verrà scelta automaticamente quando scarichi i tuoi file.

Regole dei filtri:
- I filtri sono definiti usando il nome del file e il dominio.
Attenzione: Per il dominio non usare l'inidirizzo corrente del sito, ma il server dove si trova il file.
- I filtri devono corrispondere al nome del file o al dominio. Per esempio non scrivere "google.com" ma solo google.com (senza doppi apici)
- Puoi usare i caratteri wildcard * (asterisco) per rimpiazzare parti del filtro: wiki*.org
- L'asterisco non serve all'inizio e alla fine della stringa usata come filtro. Scrivere *wiki* o wiki è la stessa cosa.
- La stessa cosa vale per il filtro sul nome file: *.zip o .zip funzionano allo stesso modo.
- I filtri sono verificati da quello più in alto a quello più basso e il processo termina quando un filtro corrisponde al file da scaricare. Abbi cura di ordinare i tuoi filtri di conseguenza.
- I filtri non sono 'case insensitive', cioè non c'è nessuna differenza tra parole scritte in maiuscolo o in minuscolo quando viene processato un filtro.


Questa estensione può essere usata con espressioni regolari.
Puoi leggere dal menu l'aiuto sulle espressioni regolari sulla finestra di aggiunta e modifica filtri.


Sono disponibili variabili dinamiche per creare directory quando si salva, chiamate con la data, col nome del file o col nome del dominio.
Puoi controllare il tab delle opzioni per vedere la lista dei tag dinamici.



Un po di esempi sui filtri:

Se vuoi che tutti i file eseguibili .exe vengano salvati su una cartella specifica:
- dominio: tutti
- filter: .exe
- cartella: la tua cartella per i file eseguibili


Se invece vuoi che tutti i file di un particolare sito (per esempio mozilla.org) vengano scaricati sulla cartella Mozilla:
- dominio: mozilla.org
- filtro: tutti
- cartella: la tua cartella Mozilla


Oppure, se hai bisogno di avere tutti i file zip .zip da mozilla.org, e lasciare gli altri file nella cartella precedente:
- dominio: mozilla
- filtro: .zip
- cartella: la tua cartella Mozilla per i file zip
e metti questo filtro sopra il precedente. Ricorda che i filtri vengono controllati dall'alto verso il basso.


Per finire, puoi definire filtri con un nome più completo, e non solo dall'estensione del file:
- dominio: tutti
- filtro: firefox*.exe
- cartella: la tua cartella per tutte le versioni del programma di istallazione di Firefox


Ecco qualche esempio di variabili dinamiche e "capture": :

Per scaricare tutte le nightly build di firefox in una sottocartella di D:\firefox :
- dominio : Tutti
- filtro : firefox*pre
- cartella : D:\firefox\Nightly\%Y%-%m%-%d%
L'estensione creerà una sottocartella chiamata con la data di quel giorno. Per esempio "2009-12-04", dove verrà salvato il file.


Per scaricare un file in base alla sua estensione:
- dominio : Tutti
- filtro : Tutti
- cartella : D:\download\%asf_x%
L'estensione creerà una sottocartella chiamata "zip" per i file .zip, "rar" per i file .rar, etc., dove verranno salvati i file.


Per creare oppure no sottocartelle differenti in base al nome del file :

Esempio : Vogliamo salvare i video chiamati "La_mia_vacanza_data-di-viaggio.avi".
I video hanno i sottotitoli nel formato .srt per spiegare la vacanza, ci sono anche file .zip con le foto del viaggio, e i video possono escarticati con l'estensione .avi, .torrent or .nzb (newsgroup).

".avi" e ".srt" andranno nella cartella D:\video\vacation .
".zip", ".torrent" e ".nzb" andranno rispettivamente nelle sottocartelle D:\video\vacanze\zip, D:\video\vacanze\torrent and D:\video\vacanze\nzb.

- dominio : Tutti
- filtro : /^La_mia_vacanza.*\.(avi|srt|zip|torrent|nzb)$/
- cartella : D:\video\vacanza\%asf_rf%(zip|torrent|nzb)$%asf_rf%

Spiegazione dei filtri :
I filtri saranno attivati per tutti i file che cominciano con "La_mia_vacanza" e che finiscono con l'estensione disponibili (avi, srt, zip, torrent o nzb).
La cartella creata dipenderà dal filtro: %asf_rf% analizza il nome del file scaricato con l'espressione regolare contenuta fra i due %asf_rf% : se il file finisce con "zip", "torrent" o "nzb", la parte catturata dall'espressione regolare creerà la sottocartella.
Se nessuna di quelle estensioni non è trovata, l'espressione regolare non farà nulla, non verrà creata nessuna sottocartella e i file verranno scaricati nella cartella in D:\video\vacanze\

Nota: Puoi mettere quanti tag vuoi per la cartella principale.
I filtri possono essere:
- filtro : /.*\.(avi|srt|zip|torrent|nzb)$/
- cartella : D:\video\%asf_rf%(vacanza|matrimonio)%asf_rf%\%asf_rf%(zip|torrent|nzb)$%asf_rf%
In questo cartello, verrà creata la cartella "vacanze" verrà creata solo se il nome contiene "vacanze".




Problemi noti:

- Per alcuni utenti, ASF suggerisce solo la cartella di default invece della cartella del filtro.
Non ci sono abbastanza commenti da parte degli uteniti per capire bene il problema.
Perfavore contattare l'autore se sei uno di tali utenti e vuoi aiutare a correggere il problema.

- Se i filtri contenevano caratteri unicode nella versione precedente (precedente alla 0.9.3), il percorso del filtro potrebbe apparire vuoto, ma funziona.
Perfavore, edita i filtri e specifica il percorso manualmente.

- I filtri dei domini sono fatti nel referer URL se il dominio hostato non corrisponde al filtro.
Può dare problemi sulla validazione del filtro quando l'utente non prova a validarlo.
Se tale problema capita, cancellerò quest'opzione (perfavore lasciate dei commenti su questa opzione: utile o no?).
